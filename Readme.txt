MSC Thesis Evaluation System




For backed:
Install xammp (v7.2.7) and run xammp server and mysql
take thesis folder to xammp/htdocs
Import database.sql from mysql admin(mysql admin in xammp control panel)




For Frontend:
Install node js(v12.18.2)
 go to thesis_system folder
install react router,react icons, bootstrap, axios
Open cmd and enter npm start
Install the extension Moseif cors in the browser and switch on the cors
